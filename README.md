## Project Overview
A 3D scene simulation project created to explore computational graphics and visualization techniques.
Focused on building a realistic tabletop environment with objects such as a laptop, coffee cup, orange, and a sprite bottle, using lighting, textures, and materials to enhance visual realism.

**Tech Used:** C++, OpenGL, GLSL Shaders, GLM, stb_image, Computational Graphics
